# 食品成分表檢查系統前端

## 項目簡介
本專案是食品公司用於判斷成分表是否出錯的系統前端，主要負責上傳影像並進行結果輸出。

## 目前進度
- **第一版主頁** 已經搭建完成，包含：
  - 左側導航欄
  - 上傳影像區塊
  - 結果顯示區塊
  - 輸出按鈕

## 技術棧
- **前端框架**：React
- **UI 框架**：Tailwind CSS
- **圖示庫**：Lucide React

## 使用方法
1. **安裝依賴**：
   ```bash
   npm install
   ```
2. **運行專案**：
   ```bash
   npm start
   ```
3. 在主頁上傳影像，系統會處理並顯示結果。
